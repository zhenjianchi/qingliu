清流是一款基于神经科学的健康习惯工具。当渴望高峰来袭（约 90 秒），我们用 循证打断技术（CBT / ACT） 帮你的前额叶重新夺回主动权；当破戒发生， 我们用科学叙事取代羞耻感；当日子往前走，我们用可验证的恢复进程让你看见变化。

视觉调性

温暖陪伴

沙色基底取代纯白，暖墨色文字取代冷黑。避开临床感和工具冷感，把"陪伴"作为视觉基调。

字体语言

SF Pro + 汉字

英文 SF Pro Display / Text，中文走 SF Pro 的汉字回退链。Display 紧 tracking 走苹果"机器感"标题，body 17px 走 Retina 阅读基线。

Aa 字

操作主色

单蓝色动作

Apple Action Blue 是唯一可靠的动作语言。在暖色背景上更具指向性，每屏只出现 ≤2 次，绝不与品牌色抢戏。

叙事原则

去羞耻化

禁用"超能力/重启/觉醒/失败"等叙事。每一条文案都用神经科学语言（多巴胺受体、前额叶、90 秒高峰）替换判断性语言。

选择你的平台

清流同时支持 iOS 与 Android。本轮交付包含 6 个核心屏幕：首页、渴望应对（最关键路径）、T4 90 秒呼吸法、破戒后支持问卷、AI 即时支持、效果数据面板。整套交互以"在 3 秒内打开能为此刻分忧的工具"为目标。

五条不能妥协的原则

从 PRD §4 拆出，把"伦理边界"翻译成设计语言：

01

破戒 ≠ 失败

所有文案禁用"失败/归零/从头来"，改用"保留累计时间 + 你学到的 + 你拥有的"。

02

神经科学语言

用"多巴胺受体 / 前额叶抑制 / 90 秒高峰"等可验证语言，禁用"超能力 / 重启 / 觉醒"。

03

3 秒内可达

从任意屏幕到渴望应对面板 ≤ 3 秒。主屏的"渴望应对"按钮直接触达推荐技术。

04

证据强度排序

T11 / T4 / T5 优先呈现，其余 8 种折叠在"查看全部"中。每条引用都标注研究来源。

05

隐私优先

AI 对话、效果周报、伙伴匹配都默认匿名 / 本地处理，开启公共分享需显式同意。

06

非临床边界

在 AI 对话、效果数据等模块明示"本产品不替代专业心理咨询"，并提供转介路径。
