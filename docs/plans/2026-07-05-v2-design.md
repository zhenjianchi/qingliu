## 1. 背景与目标

### 1.1 问题诊断

**市场反馈**："下载量/使用率低"
- 用户首次打开后的价值感知不足
- 功能虽多但感知"功能单薄"
- UI 风格陈旧，不符合年轻用户审美
- 缺乏游戏化驱动，难以形成持续使用习惯

### 1.2 目标

| 指标 | 当前 | 目标 |
|------|------|------|
| 首日留存率 | 低 | +50% |
| 7 日留存率 | 低 | +30% |
| 30 日留存率 | 低 | +20% |
| 渴望面板使用率 | 低 | +100% |
| 用户满意度 (NPS) | 不明 | >50 |

### 1.3 目标用户

- **画像**: 18-30 岁年轻人，中文戒色/脱瘾场景
- **特征**: 注意力短、视觉敏感、对陈旧设计反感、追求成就感
- **使用场景**: 戒色过程中孤独感强，需要持续激励与科学支持

## 2. 设计系统 (基于 Open Design Lingo)

### 2.1 色彩方案

| Token | Light | Dark | 用途 |
|-------|-------|------|------|
| Primary | `#58CC02` | `#7BE325` | CTA、进度、积极反馈 |
| Secondary | `#CE82FF` | `#B468E8` | 强调、装饰、平静场景 |
| Warning | `#FFC800` | `#FFD033` | 提示、能量 |
| Danger | `#FF4B4B` | `#FF6B6B` | 紧急、警示 |
| Background | `#FFFFFF` | `#1A1B2A` | 页面背景 |
| Surface | `#F7F7F7` | `#252736` | 卡片背景 |
| Text Primary | `#3C3C3C` | `#F5F5F7` | 正文 |
| Text Secondary | `#6B6B6B` | `#A0A0AC` | 辅助说明 |
| Border | `#EAEAEA` | `#3A3B4A` | 卡片描边 |

### 2.2 字体

- **Display**: Nunito (Black 800-900, 32-72pt)
- **Body**: Nunito (Regular 400 / Medium 500, 14-16pt)
- **Number**: Nunito (Black 900, 48-96pt) - 大数字计数器
- **中文**: 苹方 PingFang SC / 思源黑体 Source Han Sans

### 2.3 间距系统

```
--space-1: 4px
--space-2: 8px
--space-3: 12px
--space-4: 16px
--space-5: 20px
--space-6: 24px
--space-8: 32px
--space-12: 48px
```

### 2.4 圆角

- `--radius-sm`: 10px (按钮、输入框)
- `--radius-md`: 16px (卡片)
- `--radius-lg`: 24px (大卡片、底部弹层)
- `--radius-pill`: 9999px (胶囊标签、徽章)

### 2.5 阴影 / 立体感

- `--elev-flat`: 无阴影
- `--elev-raised`: `0 8px 24px rgba(88, 204, 2, 0.18)` - 主CTA绿色光晕
- `--elev-card`: `0 4px 12px rgba(0, 0, 0, 0.08)` - 卡片浅阴影
- `--elev-3d`: `inset 0 -4px 0 rgba(0, 0, 0, 0.15)` - 3D厚边

### 2.6 核心组件

| 组件 | 描述 | 替换 |
|------|------|------|
| `TactileCard` | 立体厚边卡片 (Duolingo 风) | 通用 Card |
| `Pill` | 圆角胶囊标签 | 通用 Chip |
| `StepCounter` | 大数字 + 渐变背景 | 圆形进度环 |
| `LevelBadge` | 等级徽章 (3D立体) | 通用 Icon |
| `DayWall` | emoji + 颜色卡片墙 | 通用日历热力图 |
| `QuoteStrip` | 每日金句滚动条 | 通用文字组件 |
| `UrgeFAB` | 渴望浮动按钮 (脉冲动画) | 通用 FAB |

### 3.2 Hero 数字区（核心）

- 巨大数字 Nunito 900 96pt，主色绿渐变背景
- 立体厚边框 (`--elev-3d`)
- 背景庆祝粒子（每 30 天触发一次大彩纸）
- 副标题科学叙事（轮播）

### 3.3 今日仪式 (Daily Ritual)

| 时段 | 内容 | 时长 |
|------|------|------|
| 早晨承诺 | 一句话宣言 + 选定今日重点技术 | <10秒 |
| 傍晚回顾 | 情绪打分 + 应对记录 + 反思 | <2分钟 |
| 夜间总结 | 当日 XP 获得 + 明日鼓励 | <5秒 |

### 3.4 DayWall 时间墙

- 30 天卡片墙（每行7个，共5行）
- 每格带 emoji（🔥强力/😊普通/😅艰难/😢低谷）
- 颜色映射心情档位
- 左右滑动切换月份
- 点击单格查看当日详情

### 3.5 神经恢复进度

- 不是环形图，是分阶段"关卡条"
- Day 7 → Day 14 → Day 30 → Day 60 → Day 90 → Day 365
- 每过一关解锁新"超能力"叙事
- 当前关卡通关动画（粒子 + 庆祝）

### 3.6 成就徽章墙

- 3D 立体徽章（Duolingo 风）
- 解锁动画：抖动 + 粒子 + 全屏庆祝
- 分类：坚持类 / 应对类 / 反思类 / 学习类 / 特殊类

### 3.7 渴望 FAB

- 醒目绿底白色加号，3D 厚边
- 持续脉冲动画（轻微缩放）
- 点击 1 秒可达渴望面板

## 4. 渴望面板重构

### 4.1 入口动效

- FAB 点击 → 全屏绿色脉冲（200ms）
- 面板从底部弹出（spring 物理动画，400ms）
- 背景模糊化（glassmorphism + 紫色调）

### 4.2 强度选择

不是 slider，是**语义化三选一**：

```
┌─────────────────────────────────┐
│  此刻你的渴望是什么感觉？         │
│                                 │
│  ┌─────────┐ ┌─────────┐ ┌──────┐│
│  │  😌     │ │  😰     │ │  🔥  ││
│  │  有点想 │ │  很强烈 │ │压不住││
│  │  (1-3)  │ │ (4-7)   │ │(8-10)││
│  └─────────┘ └─────────┘ └──────┘│
└─────────────────────────────────┘
```

### 4.3 智能推荐

基于所选强度，从 11 个技术中智能匹配：

- 顶部大卡：「先试试这个 →」
- 3D 厚边推荐技术卡（T4/T5/T11 智能匹配）
- 「换一换」按钮 → 摇一摇动画推荐下一个
- 「查看全部 11 个」折叠区

### 4.4 沉浸式引导

点击技术卡 → 全屏沉浸式引导：
- **T4 呼吸**: 气泡膨胀/收缩动画（4-2-15秒节奏）
- **T5 接地**: 分步骤卡片（5→4→3→2→1），语音+手动
- **T11 冷水+对话**: 30秒计时器 + 自我对话模板

### 4.5 认知重构升级

- 滑出抽屉式
- 输入框 + 「AI 帮你看看」按钮（本地处理）
- 「重写你的想法」模板

### 4.6 历史自动存档

每次应对自动记录：
- 时间、强度、用了什么技术、结果（成功/部分成功/失败）
- 「你已经成功应对 X 次」显示在面板顶部

## 5. 数据可视化 + 游戏化

### 5.1 神经恢复地图

简化大脑图 + 高亮区域动画：
- 多巴胺通路（红色 → 黄色 → 绿色）
- 前额叶（灰 → 蓝 → 紫）
- 每个恢复阶段配一个徽章

### 5.2 新统计页

| 模块 | 描述 |
|------|------|
| 总览卡 | 大数字网格：累计天数 / 应对次数 / 最高连续 |
| 心情热力图 | 30 天日历 + emoji 颜色 |
| 触发分析 | 饼图 + 折线图 + 文字洞察 |
| 个人里程碑 | 时间线展示 |

### 5.3 XP/Level 系统

| 行为 | XP |
|------|-----|
| 完成每日打卡 | +20 |
| 成功应对渴望 | +50 |
| 完成 T4/T5/T11 | +30 |
| 7天连续 | +200（+徽章） |
| 30天连续 | +1000（+专属徽章） |
| 触发反思记录 | +15 |

等级：
- Lv1-5: 新手（绿叶）
- Lv6-10: 觉醒（紫水晶）
- Lv11-15: 战士（金盾牌）
- Lv16+: 大师（彩虹光环）

### 5.4 成就系统（30+ 徽章）

- **坚持类**: Day 1/7/30/90/365
- **应对类**: 首次应对/100次/1000次
- **反思类**: 第一篇日记/连续30天日记
- **学习类**: 完成所有引导/学会10个技巧
- **特殊类**: 30天早睡/30天运动

## 6. Onboarding + 首次体验

### 6.1 首次启动流程（2 分钟内）

| 步骤 | 内容 | 时长 |
|------|------|------|
| 欢迎屏 | "你不是一个人在战斗" | 5s |
| 价值主张 | 3 张轮播图 | 15s |
| 目标设定 | 30/90/100天选择 + 宣言 | 30s |
| 兴趣偏好 | emoji 多选触发场景 | 20s |
| 庆祝首屏 | "Day 1 ✦ 启程" 彩纸 | 3s |

### 6.2 Demo 数据 / 引导体验

- 不展示空状态
- 完整"理想首日"动效
- 一键打卡 + 5分钟内完成仪式
- 获得第一个 XP

### 6.3 永久保留

- 关闭后下次不再出现
- 设置中可重看

## 10. 验收标准

### 10.1 设计层面
- [ ] 色彩与 Lingo 系统对齐
- [ ] 字体统一为 Nunito
- [ ] 所有 Card 有 3D 厚边或 tactile 感
- [ ] 无圆形进度环（保留语义化进度）
- [ ] 暗色模式默认

### 10.2 功能层面
- [ ] FAB 1 秒可达渴望面板
- [ ] Onboarding 2 分钟内完成
- [ ] XP/Level 实时更新
- [ ] 成就解锁动画流畅（<500ms）
- [ ] DayWall 流畅滚动

### 10.3 数据层面
- [ ] 累计天数保留
- [ ] 应对历史自动存档
- [ ] 心情数据可关联分析

## 11. 风险与缓解

| 风险 | 影响 | 缓解 |
|------|------|------|
| Nunito 字体未集成 | UI 不一致 | pubspec.yaml 添加 google_fonts |
| XP 数值不合理 | 用户反感 | 内部测试调优 |
| 3D 厚边过度使用 | 视觉疲劳 | 限制使用场景（仅 CTA 和 Hero 卡）|
| Onboarding 跳过率高 | 价值感知不足 | 可重看 + Demo 数据兜底 |
| 暗色模式色彩对比 | 可读性 | AA 标准校验 |
